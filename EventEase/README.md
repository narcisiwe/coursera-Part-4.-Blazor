# EventEase Blazor Project

EventEase is a polished Blazor Web App created for the Coursera peer-graded assignment. The project demonstrates:

- Event Card component generation with two-way binding
- Routing between multiple pages
- Debugging and performance-minded cleanup
- Registration form validation with data annotations
- Session state management using scoped services
- Attendance tracking with check-in toggles
- A project summary explaining how Microsoft Copilot helped during development

## Run locally

```bash
dotnet restore
dotnet run
```

Then open:

```text
http://localhost:5000
```

## Pages

- `/` Home dashboard
- `/events` Event Cards demo
- `/register` Registration form with validation
- `/attendance` Attendance tracker
- `/session` Session state demo
- `/copilot-summary` Copilot development summary
