Copilot Summary:
I used Microsoft Copilot throughout the development of this Blazor project. First, it helped generate the foundational Event Card component with important event fields and two-way binding. After that, it assisted me with implementing routing and navigation between different pages in the application.

Copilot also helped during debugging by suggesting fixes for routing issues, validation problems, and general code cleanup. In the final stage, it supported the implementation of advanced features such as the registration form with validation, session-based user state, and the attendance tracker. Overall, Copilot made development faster, helped organize the project structure, and improved the final quality of the app.
